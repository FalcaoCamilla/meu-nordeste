# Meu Nordeste
O Meu Nordeste, ainda em construção, surgiu a partir de um desafio da Tech Academy, da StartSe.
Utilizando apenas HTML e CSS, a aplicação objetiva valorizar os estados no Nordeste, mostrando seus principais pontos turísticos e passeios pela região.
